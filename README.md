# Local Development

## Frontend

To run the frontend, do:

```
cd zealthyfront
npm install
npm start
```

This will run the frontend on http://localhost:3000/

## Backend

To run the frontend, do:

```
cd zealthyprojbackend
npm install
npm start
```

This will run the backend on http://localhost:8080/
